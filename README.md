## Overview

Job Applications Tracker is built to simplify the process of organizing job applications. With a responsive layout and intuitive controls, users can monitor application status, update details in real-time, and export data when needed. No installation or backend required — everything runs directly in the browser with persistent local storage.

---

## Key Features

- Smart save system with autosave and manual save options  
- Real-time statistics for total, pending, applied, and responded applications  
- Excel export functionality for external record-keeping  
- Clean, responsive, and modern user interface  
- Add, edit, and delete applications with ease  
- Automatic data persistence through browser storage  
- Real-time updates for status and summary counts

## Getting Started

### Requirements
- A modern web browser (Chrome, Firefox, Edge, Safari)  
- No installation, server, or backend needed  

### How to Use
1. Open the `index.html` file in your browser.  
2. Add new applications through the "Add Application" button.  
3. Edit any field directly (platform, website, dates, notes).  
4. Change status using the dropdown menu.  
5. Click "Save Changes" to store your updates.  
6. Export all records to Excel using the "Export to Excel" button.  

---

## Detailed Features

### Application Management
- Add, edit, and delete entries  
- Inline editing for fast updates  
- Automatic row numbering  
- Confirmation before deletion  

### Status Categories
- Pending: Applications you plan to submit  
- Applied: Applications already submitted  
- Responded: Applications with replies  

### Data Protection
- Autosave every 30 seconds  
- Manual save option for instant updates  
- Unsaved changes indicator  
- Browser warning before closing with unsaved data  
- Data stored locally for long-term access  

### Exporting
- Export complete data as an Excel (.xlsx) file  
- Automatic file naming with current date  
- Includes all fields and statuses
